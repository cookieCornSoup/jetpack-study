## 컴포저블 관련 기능

### @Preview

* Preview 어노테이션은 Composable 부분을 IDE에서 미리보기로 볼 수 있음
* [관련 문서](https://developer.android.com/jetpack/compose/preview?hl=ko)

```kotlin
    @Preview
    @Composable
    fun Greeting(name: String){
        Text(text = "Hello $name!")
    }
```
